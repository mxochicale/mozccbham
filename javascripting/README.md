# [javascripting](https://github.com/workshopper/javascripting)

```
sudo npm install -g javascripting
```
-g: means global


# USAGE
Open your terminal and run the following command:
```
javascripting
```
and then to verify
```
javascripting verify script.js
```

or to run
```
javascripting run script.js
```
